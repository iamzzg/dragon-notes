# CMD 命令

## 查看某个命令的用方法

> 命令 /?

## 强制删除文件夹及其下所有目录和文件 rmdir,rd

```bash
# /s: 表示所有文件,/q: 表示不用询问
rmdir /s /q 文件夹名

rd /s /q 文件夹名
```

## 删除某个文件

```bash
del 文件名
```

## 创建文件夹

```bash
mkdir 文件夹名
md 文件夹名
```

## 创建文件(带内容 echo "" >和不带 cd.>)

```bash
# 带内容
echo 内容 -> test.txt
# 不带内容
cd.> test.txt
```
