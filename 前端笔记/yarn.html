# yarn

# 常用命令

  1. 显示命令列表
yarn help
  1. 初始化一个项目
yarn init
yarn init
  1. 安装所有依赖项
yarn
yarn install
  1. 添加依赖项
yarn add [package]
yarn add [package]@[version]
yarn add [package]@[tag]
  1. 将依赖项添加到不同的依赖类别中
yarn add [package] --dev
yarn add [package] --peer
  1. 更新依赖项
yarn up [package]
yarn up [package]@[version]
yarn up [package]@[tag]
  1. 删除依赖项
yarn remove [package]
  1. 运行 scripts 脚本
yarn 脚本命令
